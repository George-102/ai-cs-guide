    吴恩达 Machine Learning

    课程简介

    • 来源：Stanford / Coursera（Andrew Ng）
    • 先修要求：Python 基础、线性代数、概率论
    • 编程语言：Python（NumPy）
    • 课程难度：🌟🌟🌟
    • 预计学时：60 小时 +

    吴恩达的 Machine Learning 课程是全球最受欢迎的 ML 入门课程，没有之一。Andrew Ng 是 AI 领域的传奇人物——Google Brain 创始人、Coursera 创始人、前百度 AI 负责人。

    这门课的最大特点是讲解极其清晰。Andrew Ng 能把复杂的数学概念用直觉化的方式讲清楚，让零基础的学生也能理解。

    课程资源

    • 课程网站：Coursera（可免费旁听）
    • 新版课程：Machine Learning Specialization（3 门课，2022 年更新）
    • 课程笔记：CS229（Stanford 原版课程）

    课程内容

    第一部分：监督学习

    • 线性回归（单变量、多变量）
    • 逻辑回归
    • 正则化（L1、L2）
    • 神经网络基础

    第二部分：无监督学习

    • K-Means 聚类
    • 主成分分析（PCA）
    • 异常检测

    第三部分：应用

    • 推荐系统
    • 大规模机器学习

    自学建议

    做所有的编程作业。 课程用 Python/NumPy 实现算法，不调库。这是最宝贵的学习体验。

    不要跳过数学推导。 Andrew Ng 的数学推导非常清晰，跟着推一遍比看十遍视频都管用。

    配合 Kaggle 练习。 学完一个算法，去 Kaggle 上找对应的数据集练手。

    备注

    这门课是 ML 入门的最佳选择。如果你只想学一门 ML 课程，选这门就够了。

    新版课程（Machine Learning Specialization）用 Python 实现，比旧版（Octave/MATLAB）更实用。
