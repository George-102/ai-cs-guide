# Princeton Algorithms — 普林斯顿算法课

## 简介

Princeton Algorithms 是普林斯顿大学 Robert Sedgewick 和 Kevin Wayne 在 Coursera 上开设的算法课程，分为两部分：Part I 涵盖基础数据结构和排序，Part II 涵盖图算法和字符串处理。

这门课最大的特点是**与《算法（第4版）》教材完美配套**。Sedgewick 既是课程主讲人，也是教材作者，所以课程内容和教材高度一致。课程使用 Java 实现所有算法，并提供了大量可视化和交互式演示，让抽象的算法变得直观可感。

课程在 Coursera 上提供，有自动评分的编程作业。作业设计非常精妙 — 比如用双端队列实现一个滑动窗口最大值算法，用 KD-Tree 实现一个范围搜索。

## 先修要求

- **Java 基础**：课程使用 Java 实现所有算法，需要熟悉 Java 语法
- **编程经验**：至少完成一门编程入门课程
- **不需要**：算法或数据结构的前置知识

## 难度

🌟🌟⭐（中等）

Part I 相对容易，Part II 的图算法和字符串处理难度上升。编程作业有一定挑战性，但课程提供了充分的提示和测试用例。

## 学时

约 60-80 小时（Part I + Part II 合计）

- Part I：约 30-40 小时
- Part II：约 30-40 小时

## 自学建议

1. **先看书再看视频。** Sedgewick 的书写得非常清晰，可以先读一章对应的教材内容，再看视频加深理解。

2. **一定要做编程作业。** Coursera 的编程作业有自动评分系统，能即时反馈你的实现是否正确。作业的设计往往比考试更有教育价值。

3. **善用课程的可视化工具。** Princeton 提供了大量算法可视化演示，对理解排序、树操作、图遍历等算法的执行过程非常有帮助。

4. **Part II 的字符串处理值得重点学习。** KMP、Rabin-Karp、Trie、后缀数组等字符串算法在实际工程中应用广泛，但很多课程不讲。

5. **如果时间有限，优先完成 Part I。** Part I 覆盖了面试中最常考的数据结构和算法，性价比最高。

6. **配合 LeetCode 刷题。** 每学完一个专题，去 LeetCode 上找 5-10 道对应题目巩固。
