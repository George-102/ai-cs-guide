# Computer Networking: A Top-Down Approach

## Descriptions

- Source: James Kurose, Keith Ross (University of Massachusetts)
- Prerequisites: None, suitable for beginners
- Difficulty: 🌟🌟🌟
- Class Hour: 60+

*Computer Networking: A Top-Down Approach* is one of the most popular networking textbooks in the world. Its unique feature: **starting from the application layer** and working down to the lower layers — exactly matching how you experience the network.

The book is known for **clear explanations, rich examples, and comprehensive resources**. Each chapter has extensive exercises, programming assignments, and Wireshark labs.

## Resources

- Official site: [gaia.cs.umass.edu/kurose_ross/](https://gaia.cs.umass.edu/kurose_ross/)
- Resources: Lecture slides, Wireshark labs, programming assignments
- Chinese translation: 《计算机网络：自顶向下方法》(7th Edition)

## Content Structure

### Application Layer
- HTTP, DNS, SMTP, P2P
- Socket programming

### Transport Layer
- UDP, TCP
- Reliable data transfer, congestion control

### Network Layer
- IP protocol, routing algorithms
- IPv4/IPv6

### Link Layer
- Ethernet, switches
- ARP, VLAN

### Network Security
- Cryptography basics
- TLS/SSL, firewalls

## Self-Study Tips

**Start from the application layer.** This is the book's signature approach — begin with HTTP and DNS (what you know), then go deeper.

**Do the Wireshark labs.** Each chapter has accompanying Wireshark labs that let you see real network packets.

**Pair with Stanford CS144.** This book covers theory; CS144 provides hands-on labs. Together, they're the perfect combination.

## Notes

This is the best textbook for networking beginners. If you only read one networking book, this should be it.
