# Stanford CS144: Introduction to Networking

## Descriptions

- Offered by: Stanford University
- Prerequisites: Computer architecture basics, C++ basics
- Programming Language: C++
- Difficulty: 🌟🌟🌟🌟
- Website: [cs144.github.io](https://cs144.github.io/)

Stanford CS144 is Stanford's networking course, distinguished by its **lab-driven approach**. You implement a complete TCP/IP protocol stack — from the network interface layer to the transport layer — and your implementation must pass real network communication tests.

## Self-Study Tips

**Solid C++ skills required.** The course uses C++; familiarity with pointers, memory management, and OOP is essential.

**Do labs in order.** Each lab depends on the previous lab's code; you can't skip ahead.

**Verify with Wireshark.** After each lab, use Wireshark to capture packets and verify your implementation is correct.

## Notes

CS144 has some of the best lab design in networking courses. After completing all labs, your understanding of TCP/IP will surpass most practitioners.
