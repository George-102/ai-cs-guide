# 计算机网络：自顶向下方法

## 教材简介

- 来源：James Kurose、Keith Ross（University of Massachusetts）
- 先修要求：无特殊要求，适合入门
- 课程难度：🌟🌟🌟
- 预计学时：60 小时 +

《计算机网络：自顶向下方法》（Computer Networking: A Top-Down Approach）是全球最受欢迎的计算机网络教材之一。它的独特之处在于**从应用层开始讲起**，逐步深入到底层——这和你使用网络的体验完全一致。

这本书的特点是：**讲解清晰、案例丰富、配套资源完善**。每章都有大量的练习题和编程作业，还有 Wireshark 实验。

## 教材资源

- 官方网站：[gaia.cs.umass.edu/kurose_ross/](https://gaia.cs.umass.edu/kurose_ross/)
- 配套资源：课程 PPT、Wireshark 实验、编程作业
- 中文翻译：《计算机网络：自顶向下方法》（第 7 版）

## 内容结构

### 应用层
- HTTP、DNS、SMTP、P2P
- Socket 编程

### 传输层
- UDP、TCP
- 可靠数据传输、拥塞控制

### 网络层
- IP 协议、路由算法
- IPv4/IPv6

### 链路层
- 以太网、交换机
- ARP、VLAN

### 网络安全
- 密码学基础
- TLS/SSL、防火墙

## 自学建议

**从应用层开始。** 这是这本书的特色——先讲你最熟悉的 HTTP 和 DNS，再逐步深入底层。

**做 Wireshark 实验。** 每章都有配套的 Wireshark 实验，能让你看到真实的网络数据包。

**配合 Stanford CS144 使用。** 这本书讲理论，CS144 做实验。两者结合效果最好。

## 备注

这本书是计算机网络入门的最佳教材。如果你只想读一本网络教材，选这本就够了。
