Python 是一种**解释型、面向对象的高级编程语言**，由吉多·范罗苏姆于1991年发布。它以**简洁优雅的语法**著称，用缩进代替大括号来定义代码块，让程序读起来像英语一样直观。Python的设计哲学是“**优雅、明确、简单**”，强调代码的可读性和开发效率。它拥有极其丰富的标准库和第三方生态，覆盖网络爬虫、数据分析、人工智能、Web开发、自动化运维等几乎所有领域。
