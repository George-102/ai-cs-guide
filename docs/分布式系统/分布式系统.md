# 分布式系统

## 什么是分布式系统

分布式系统研究的是**多台计算机如何协同工作来完成单台计算机无法完成的任务**。你使用的几乎所有大型互联网服务——搜索引擎、社交网络、电商、云存储——背后都是分布式系统。

分布式系统的核心挑战在于：网络不可靠（消息可能丢失、延迟、乱序）、节点可能故障、时钟不同步、数据需要在多个副本间保持一致。分布式系统的设计就是在这一个个"不可能"之间寻找可行的折中。

## 为什么要学分布式系统

**大规模系统的必修课。** 当代 AI 训练（GPT、Gemini）需要数千块 GPU 协同工作，AI 推理服务需要在全球多个数据中心部署——这些都是分布式系统问题。

**理解现代软件架构。** 微服务、容器编排（Kubernetes）、消息队列、分布式数据库——这些你工作中必然用到的技术，底层都是分布式系统的理论支撑。

**面试高阶考点。** 系统设计面试（System Design Interview）的核心就是分布式系统设计。CAP 定理、一致性协议、负载均衡——这些是大厂高阶岗位的必考内容。

**AI 基础设施的关键。** 分布式训练（数据并行、模型并行、流水线并行）、模型服务弹性伸缩、边缘 AI 部署——AI 工程师需要理解这些分布式计算模式。

## 核心知识模块

### 分布式系统基础

- **定义与特征**：资源共享、并发、无全局时钟、独立故障
- **系统架构**：客户端-服务器、P2P、微服务架构
- **网络基础**：RPC、消息传递、TCP/IP 回顾

### 时间与时钟

- **时钟同步**：物理时钟、NTP 协议
- **逻辑时钟**：Lamport 时钟、向量时钟
- **全局状态**：Chandy-Lamport 快照算法

### 复制与一致性

- **复制策略**：主从复制、多主复制、无主复制
- **一致性模型**：强一致性、最终一致性、因果一致性
- **CAP 定理**：一致性、可用性、分区容忍性的不可能三角
- **BASE 理论**：Basically Available、Soft state、Eventually consistent

### 共识算法

- **Paxos**：分布式共识的经典算法（难懂但重要）
- **Raft**：Paxos 的"可理解版本"，工业界广泛使用
- **ZAB**：ZooKeeper 使用的原子广播协议
- **两阶段提交（2PC）和三阶段提交（3PC）**

### 分布式数据存储

- **数据分区（Sharding）**：哈希分区、范围分区、一致性哈希
- **分布式数据库**：Google Spanner、CockroachDB、TiDB
- **分布式文件系统**：GFS、HDFS
- **分布式缓存**：Redis Cluster、Memcached

### 分布式计算

- **MapReduce**：大规模数据处理的编程模型
- **Spark**：内存计算引擎，MapReduce 的进化
- **分布式训练**：数据并行、模型并行、流水线并行

### 容错与可靠性

- **故障模型**：崩溃故障、拜占庭故障
- **复制与容错**：状态机复制
- **分布式事务**：Saga 模式、TCC

## 怎么学

**先读经典论文。**  Distributed Systems 的成长离不开经典论文。从 Google 三驾马车（GFS、MapReduce、Bigtable）开始，再到 Raft、Spanner——每篇论文都解决了一个真实的分布式问题。

**做 MIT 6.824 实验。** 这门课要求你实现 Raft 共识算法、基于 Raft 的 KV 存储。做完这些实验，你对分布式系统的理解会超越大多数从业者。

**从一致性哈希开始理解。** 一致性哈希是分布式系统中最优雅的概念之一，也是理解数据分片、负载均衡的钥匙。

**用 Docker Compose 搭一个分布式集群。** 自己动手部署一个 Redis Cluster 或 Kafka 集群，观察节点故障时系统如何恢复。

**教材**

- 《分布式系统：概念与设计》（Coulouris 等）— 经典教材，内容全面
- 《数据密集型应用系统设计》（Martin Kleppmann）— 最好的分布式系统入门书，强烈推荐
- 《Distributed Systems: Principles and Paradigms》（Tanenbaum）— 经典教材

**课程**

- [MIT 6.824: Distributed Systems](https://pdos.csail.mit.edu/6.824/) — 分布式系统神级课程，实现 Raft、KV 存储，强烈推荐

- [6.824 中文翻译](https://mit-public-translators.github.io/6-824/) — MIT 6.824 的中文社区翻译

**必读论文**

- [MapReduce: Simplified Data Processing on Large Clusters](https://research.google/pubs/pub62/) — Google MapReduce 原论文
