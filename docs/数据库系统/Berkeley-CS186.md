# Berkeley CS186: Introduction to Database Systems

## 课程简介

- 来源：UC Berkeley
- 先修要求：CS61B（数据结构）、CS61C（体系结构）同等水平
- 编程语言：Java
- 课程难度：🌟🌟🌟
- 课程网站：[cs186berkeley.net](https://cs186berkeley.net/)

Berkeley CS186 是 Berkeley 的数据库系统课程，覆盖关系模型、SQL、存储与索引、查询处理、事务处理、并发控制等核心主题。

## 课程资源

- 课程网站：[cs186berkeley.net](https://cs186berkeley.net/)
- 课程视频：[YouTube](https://www.youtube.com/playlist?list=PLhMnuBfGeCDPZ28Hv0FY9w_3Eo1DtqHQF)

## 课程特点

- **Java 实现**：用 Java 实现数据库组件，比 C++ 更易上手
- **项目驱动**：通过多个项目逐步构建一个简单的数据库
- **覆盖全面**：从存储层到查询层到事务层

## 自学建议

**先学 Java 基础。** 课程用 Java 实现，需要熟悉 Java 集合框架和面向对象。

**做所有的项目。** 每个项目都帮助你理解数据库的一个核心组件。

## 备注

CS186 是数据库入门的优秀课程。如果你想用 Java 实现数据库组件，这门课是最佳选择。
