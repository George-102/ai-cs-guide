# Berkeley CS186: Introduction to Database Systems

## Descriptions

- Offered by: UC Berkeley
- Prerequisites: CS61B (Data Structures), CS61C (Architecture) or equivalent
- Programming Language: Java
- Difficulty: 🌟🌟🌟
- Website: [cs186berkeley.net](https://cs186berkeley.net/)

Berkeley CS186 is Berkeley's database systems course, covering relational models, SQL, storage and indexing, query processing, transaction processing, and concurrency control.

## Course Resources

- Website: [cs186berkeley.net](https://cs186berkeley.net/)
- Lectures: [YouTube](https://www.youtube.com/playlist?list=PLhMnuBfGeCDPZ28Hv0FY9w_3Eo1DtqHQF)

## Course Features

- **Java implementation**: Easier to work with than C++
- **Project-driven**: Build a simple database through multiple projects
- **Comprehensive coverage**: From storage to query to transaction layers

## Self-Study Tips

**Learn Java basics first.** The course uses Java; familiarity with collections and OOP is required.

**Do all projects.** Each project helps you understand a core database component.

## Notes

CS186 is an excellent course for getting started with databases. If you want to implement database components in Java, this is the best choice.
