# Hung-Yi Lee Machine Learning

## Descriptions

- Source: National Taiwan University (Hung-Yi Lee)
- Prerequisites: Python basics, linear algebra, probability
- Programming Language: Python
- Difficulty: 🌟🌟🌟
- Class Hour: 50+

Hung-Yi Lee is a professor at NTU. His ML course is the highest-quality Chinese ML course available. It's updated yearly, covering both traditional ML and cutting-edge deep learning.

## Course Resources

- Lectures: [Bilibili](https://www.bilibili.com/video/BV1JE411g7XF/)
- Website: [speech.ee.ntu.edu.tw](https://speech.ee.ntu.edu.tw/~hylee/ml/2023-spring.php)
- Assignments: Updated yearly, available on course website

## Course Features

- **Chinese lectures**: Fluent language with clear concept explanations
- **Cutting-edge content**: Covers Transformers, GANs, BERT, GPT, and more
- **Updated yearly**: Content continuously updated to follow AI frontier
- **Rich assignments**: Each topic has accompanying programming assignments

## Self-Study Tips

**Pair with Andrew Ng.** Andrew Ng covers foundations; Hung-Yi Lee covers cutting-edge topics. They complement each other.

**Do the course assignments.** Hung-Yi Lee's assignments are well-designed for deep understanding.

**Watch the latest year's lectures.** Content is updated annually; watch the most recent version.

## Notes

Hung-Yi Lee's ML course is the best choice for Chinese-language ML learning. If you want to learn ML in Chinese, this is the top pick.
