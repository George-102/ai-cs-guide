# Bishop: Pattern Recognition and Machine Learning

## Descriptions

- Source: Christopher Bishop (Microsoft Research)
- Prerequisites: Linear algebra, probability, calculus
- Difficulty: 🌟🌟🌟🌟
- Class Hour: 80+

Bishop's *Pattern Recognition and Machine Learning* (PRML) is a classic ML textbook that takes a **probabilistic perspective** throughout.

The book's hallmark: viewing all ML algorithms through the lens of probability theory. Linear regression, logistic regression, neural networks, SVMs — in Bishop's framework, they're all special cases of probabilistic models.

## Resources

- Official site: [Microsoft Research](https://www.microsoft.com/en-us/research/people/cmbishop/prml-book/)
- Free PDF: Available on official website
- Course: [Cambridge ML Course](https://mlg.eng.cam.ac.uk/zoubin/course03/)

## Content Structure

### Part 1: Probability Foundations
- Probability distributions
- Linear regression (Bayesian perspective)
- Linear classification

### Part 2: Core Models
- Neural networks
- Kernel methods (SVM)
- Sparse kernel machines

### Part 3: Advanced Topics
- Mixture models and EM
- Approximate inference
- Sampling methods
- Continuous latent variables (PCA, factor analysis)
- Combinatorial models

## Self-Study Tips

**Requires strong math foundations.** PRML has high math requirements: probability, linear algebra, and calculus.

**Pair with Andrew Ng.** Build intuition with Andrew Ng first, then go deep with PRML.

**Focus on the first 5 chapters.** They cover core ML concepts; later chapters can be read selectively.

## Notes

PRML is one of the best textbooks for advanced ML theory. Essential if you want to deeply understand the probabilistic foundations of ML.

However, it's not for complete beginners. Start with Andrew Ng, then read PRML.
