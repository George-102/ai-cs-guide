# 机器学习概述

## 什么是机器学习

机器学习是 AI 的核心分支，研究如何让计算机从数据中**自动学习规律**，而不是通过显式编程来完成任务。你每天使用的推荐系统、垃圾邮件过滤、语音识别——背后都是机器学习。

## 为什么要学机器学习

**AI 工程师的基本功。** 深度学习、强化学习、大模型都建立在机器学习的基础概念之上。

**面试核心。** 偏差-方差权衡、过拟合、梯度下降——这些是 AI 面试的高频考点。

**理解 AI 系统的基础。** 不理解 ML，你只能调 API；理解了 ML，你能判断模型为什么有效、为什么失效。

## 核心知识模块

### 基础概念
- 监督学习 vs 无监督学习 vs 半监督学习 vs 自监督学习
- 训练集 / 验证集 / 测试集
- 过拟合与欠拟合
- 偏差-方差权衡
- 交叉验证
- 评估指标：准确率、精确率、召回率、F1、AUC-ROC

### 经典算法
- 线性回归、逻辑回归
- 决策树、随机森林、XGBoost
- SVM、KNN、朴素贝叶斯
- K-Means 聚类、PCA

### 模型调优
- 正则化：L1、L2、Dropout
- 特征工程
- 超参数调优
- 集成方法

## 怎么学

**先学吴恩达，再做 Kaggle。** 吴恩达的课程帮你建立扎实的理论基础，Kaggle 帮你把理论转化为实践能力。

**用 NumPy 从零实现算法。** 不要只调 scikit-learn 的 API。

**按专题学习。** 先学线性模型，再学树模型，再学聚类。
