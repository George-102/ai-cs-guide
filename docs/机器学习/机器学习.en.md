# Machine Learning

## Introduction

- Source: Andrew Ng's Machine Learning (Coursera), Hung-Yi Lee's Machine Learning (National Taiwan University)
- Prerequisites: Python basics, linear algebra, probability and statistics
- Programming language: Python (NumPy, scikit-learn)
- Course difficulty: 🌟🌟🌟
- Estimated study hours: 40+

Machine Learning is a core branch of AI that studies how computers can **automatically learn patterns from data** rather than being explicitly programmed. The recommendation systems, spam filters, speech recognition, and face recognition you use every day — all powered by machine learning.

This course is your **first gateway** into AI. Deep learning, reinforcement learning, and large language models are all built on foundational ML concepts. Without understanding bias-variance tradeoff, overfitting, and gradient descent, you'll struggle in every subsequent course.

## Core Knowledge Modules

### Fundamental Concepts

- **Supervised vs. unsupervised vs. semi-supervised vs. self-supervised learning**
- **Training / validation / test set** splitting
- **Overfitting and underfitting**
- **Bias-Variance Tradeoff**
- **Cross-Validation**
- **Evaluation metrics**: Accuracy, precision, recall, F1, AUC-ROC

### Classic Algorithms

**Supervised learning:**
- Linear regression, logistic regression
- Decision trees, random forests, gradient boosting (XGBoost, LightGBM)
- Support Vector Machines (SVM)
- K-Nearest Neighbors (KNN)
- Naive Bayes

**Unsupervised learning:**
- K-Means clustering
- Hierarchical clustering
- Principal Component Analysis (PCA)
- Anomaly detection

**Neural network foundations:**
- Perceptron, Multi-Layer Perceptron (MLP)
- Backpropagation algorithm
- Activation functions (ReLU, Sigmoid, Tanh)

### Model Tuning

- **Regularization**: L1 (Lasso), L2 (Ridge), Dropout
- **Feature engineering**: Feature selection, scaling, crossing
- **Hyperparameter tuning**: Grid search, random search, Bayesian optimization
- **Ensemble methods**: Bagging, Boosting, Stacking

## How to Learn

**Start with Andrew Ng, then do Kaggle.** Andrew Ng's course builds solid theoretical foundations; Kaggle transforms theory into practical skills. The combination works best.

**Don't skip the math.** The closed-form solution for linear regression, convergence of gradient descent, duality in SVMs — these derivations seem tedious but are key to understanding the essence of algorithms.

- **Implement algorithms from scratch with NumPy.** Don't just call scikit-learn APIs. Hand-write linear regression, decision trees, and K-Means in NumPy — you'll gain a completely different understanding.

**Learn by topic, not by textbook order.** Study linear models first (regression + classification), then tree models, then clustering, then neural network basics. Pair each topic with a corresponding Kaggle project.

## Notes

Machine learning is a fundamental skill for AI engineers. Even if you primarily use deep learning and LLMs today, ML foundations still matter — because in many real-world scenarios (tabular data, recommendation systems, risk control), XGBoost and random forests remain the optimal solution. Don't skip traditional ML just because deep learning is trendy.
