# 李宏毅 机器学习

## 课程简介

- 来源：台湾大学（李宏毅）
- 先修要求：Python 基础、线性代数、概率论
- 编程语言：Python
- 课程难度：🌟🌟🌟
- 预计学时：50 小时 +

李宏毅是台湾大学的教授，他的机器学习课程是中文 ML 课程中质量最高的。课程每年更新，内容涵盖传统 ML 和深度学习前沿。

## 课程资源

- 课程视频：[Bilibili](https://www.bilibili.com/video/BV1JE411g7XF/)
- 课程网站：[speech.ee.ntu.edu.tw](https://speech.ee.ntu.edu.tw/~hylee/ml/2023-spring.php)
- 课程作业：每年更新，可在课程网站下载

## 课程特点

- **中文讲解**：语言流畅，概念解释清晰
- **内容前沿**：涵盖 Transformer、GAN、BERT、GPT 等前沿内容
- **每年更新**：课程内容持续更新，紧跟 AI 前沿
- **作业丰富**：每个专题都有配套的编程作业

## 自学建议

**配合吴恩达课程使用。** 吴恩达讲基础，李宏毅讲前沿。两者互补。

**做课程作业。** 李宏毅的课程作业设计得很好，能帮你深入理解每个算法。

**关注最新年份的课程。** 课程内容每年更新，建议看最新一年的版本。

## 备注

李宏毅的 ML 课程是中文 ML 课程的最佳选择。如果你想用中文学 ML，这门课是首选。
