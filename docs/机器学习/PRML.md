# Bishop: Pattern Recognition and Machine Learning

## 教材简介

- 来源：Christopher Bishop（Microsoft Research）
- 先修要求：线性代数、概率论、微积分
- 课程难度：🌟🌟🌟🌟
- 预计学时：80 小时 +

Bishop 的《Pattern Recognition and Machine Learning》（PRML）是机器学习领域的经典教材，以**概率视角**贯穿全书。

这本书的特点是：从概率论的角度统一看待各种 ML 算法。线性回归、逻辑回归、神经网络、SVM——在 Bishop 的框架下，它们都是概率模型的特例。

## 教材资源

- 官方网站：[Microsoft Research](https://www.microsoft.com/en-us/research/people/cmbishop/prml-book/)
- 免费 PDF：可在官方网站下载
- 配套课程：[Cambridge ML Course](https://mlg.eng.cam.ac.uk/zoubin/course03/)

## 内容结构

### 第一部分：概率基础
- 概率分布
- 线性回归（贝叶斯视角）
- 线性分类

### 第二部分：核心模型
- 神经网络
- 核方法（SVM）
- 稀疏核机

### 第三部分：高级主题
- 混合模型和 EM 算法
- 近似推断
- 采样方法
- 连续潜在变量（PCA、因子分析）
- 组合模型

## 自学建议

**需要扎实的数学基础。** PRML 的数学要求较高，需要熟悉概率论、线性代数和微积分。

**配合吴恩达课程使用。** 先通过吴恩达建立直觉，再读 PRML 深入理论。

**重点关注前 5 章。** 前 5 章覆盖了 ML 的核心概念，后面的章节可以根据兴趣选读。

## 备注

PRML 是 ML 理论进阶的最佳教材之一。如果你想深入理解 ML 的概率基础，这本书是必读的。

但 PRML 不适合零基础入门。建议先学吴恩达课程，再读 PRML。
