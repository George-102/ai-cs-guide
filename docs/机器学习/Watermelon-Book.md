# 周志华《机器学习》（西瓜书）

## 教材简介

- 来源：周志华（南京大学）
- 先修要求：线性代数、概率论、微积分
- 课程难度：🌟🌟🌟
- 预计学时：60 小时 +

周志华教授的《机器学习》（俗称"西瓜书"）是国内最经典的 ML 教材。书名来源于作者用西瓜数据作为贯穿全书的案例。

这本书覆盖了机器学习的核心算法和理论，内容全面但偏理论。适合作为系统学习 ML 理论的教材，但不适合零基础入门。

## 内容结构

### 第一部分：基础知识
- 模型评估与选择
- 线性模型
- 决策树

### 第二部分：核心算法
- 神经网络
- 支持向量机
- 贝叶斯分类器
- 集成学习

### 第三部分：高级主题
- 聚类
- 降维与度量学习
- 特征选择与稀疏学习
- 计算学习理论

## 自学建议

**配合南瓜书使用。** 西瓜书的公式推导比较简略，南瓜书提供了详细的公式推导。

**先学吴恩达再读西瓜书。** 西瓜书偏理论，建议先通过吴恩达课程建立直觉，再读西瓜书深入理论。

**做书后习题。** 每章都有习题，能帮你检验理解程度。

## 备注

西瓜书是国内 ML 考研和面试的参考书。如果你想系统学习 ML 理论，这本书是必读的。

但西瓜书不适合零基础入门。建议先学吴恩达课程，再读西瓜书。
