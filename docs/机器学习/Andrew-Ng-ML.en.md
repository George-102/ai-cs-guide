# Andrew Ng Machine Learning

## Descriptions

- Source: Stanford / Coursera (Andrew Ng)
- Prerequisites: Python basics, linear algebra, probability
- Programming Language: Python (NumPy)
- Difficulty: 🌟🌟🌟
- Class Hour: 60+

Andrew Ng's Machine Learning course is the most popular ML course in the world, period. Andrew Ng is a legend in AI — founder of Google Brain, co-founder of Coursera, former head of Baidu AI.

The course's greatest strength is its **exceptional clarity**. Andrew Ng explains complex math concepts intuitively, making them accessible to complete beginners.

## Course Resources

- Website: [Coursera](https://www.coursera.org/learn/machine-learning) (free to audit)
- New version: [Machine Learning Specialization](https://www.coursera.org/specializations/machine-learning-introduction) (3 courses, updated 2022)
- Lecture notes: [CS229](http://cs229.stanford.edu/) (Stanford original)

## Course Content

### Part 1: Supervised Learning
- Linear regression (single and multivariate)
- Logistic regression
- Regularization (L1, L2)
- Neural network basics

### Part 2: Unsupervised Learning
- K-Means clustering
- Principal Component Analysis (PCA)
- Anomaly detection

### Part 3: Applications
- Recommendation systems
- Large-scale machine learning

## Self-Study Tips

**Do all programming assignments.** The course implements algorithms in Python/NumPy without libraries. This is the most valuable learning experience.

**Don't skip the math derivations.** Andrew Ng's math is exceptionally clear; following the derivations is more effective than watching videos ten times.

**Practice on Kaggle.** After learning each algorithm, find a corresponding dataset on Kaggle to practice.

## Notes

This is the best choice for ML beginners. If you only take one ML course, take this one.

The new version (Machine Learning Specialization) uses Python, making it more practical than the old version (Octave/MATLAB).
