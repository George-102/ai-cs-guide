# 机器学习

## 简介

- 来源：吴恩达《Machine Learning》（Coursera）、李宏毅《机器学习》（台湾大学）
- 先修要求：Python 基础、线性代数、概率论与数理统计
- 编程语言：Python（NumPy、scikit-learn）
- 课程难度：🌟🌟🌟
- 预计学时：40 小时 +

机器学习（Machine Learning）是 AI 的核心分支，研究如何让计算机从数据中**自动学习规律**，而不是通过显式编程来完成任务。你每天使用的推荐系统、垃圾邮件过滤、语音识别、人脸识别——背后都是机器学习。

这门课是你进入 AI 领域的**第一道门槛**。深度学习、强化学习、大模型都建立在机器学习的基础概念之上。不理解偏差-方差权衡、过拟合、梯度下降，后面的课程会处处碰壁。

## 核心知识模块

### 基础概念

- **监督学习 vs 无监督学习 vs 半监督学习 vs 自监督学习**
- **训练集 / 验证集 / 测试集**的划分
- **过拟合（Overfitting）与欠拟合（Underfitting）**
- **偏差-方差权衡（Bias-Variance Tradeoff）**
- **交叉验证（Cross-Validation）**
- **评估指标**：准确率、精确率、召回率、F1、AUC-ROC

### 经典算法

**监督学习：**
- 线性回归、逻辑回归
- 决策树、随机森林、梯度提升树（XGBoost、LightGBM）
- 支持向量机（SVM）
- K 近邻（KNN）
- 朴素贝叶斯

**无监督学习：**
- K-Means 聚类
- 层次聚类
- 主成分分析（PCA）
- 异常检测

**神经网络基础：**
- 感知机、多层感知机（MLP）
- 反向传播算法
- 激活函数（ReLU、Sigmoid、Tanh）

### 模型调优

- **正则化**：L1（Lasso）、L2（Ridge）、Dropout
- **特征工程**：特征选择、特征缩放、特征交叉
- **超参数调优**：网格搜索、随机搜索、贝叶斯优化
- **集成方法**：Bagging、Boosting、Stacking

## 怎么学

**先学吴恩达，再做 Kaggle。** 吴恩达的课程帮你建立扎实的理论基础，Kaggle 帮你把理论转化为实践能力。两者结合，效果最好。

**不要跳过数学。** 线性回归的闭式解、梯度下降的收敛性、SVM 的对偶问题——这些数学推导看起来烦，但它们是理解算法本质的关键。

**用 NumPy 从零实现算法。** 不要只调 scikit-learn 的 API。用 NumPy 手写一次线性回归、决策树、K-Means，你会对这些算法有完全不同的理解。

**按专题学习，不要按教材顺序。** 先学线性模型（回归+分类），再学树模型，再学聚类，最后学神经网络基础。每个专题都配合 Kaggle 上的对应项目。

## 备注

机器学习是 AI 工程师的基本功。即使你现在主要用深度学习和大模型，ML 基础仍然重要——因为很多实际场景（表格数据、推荐系统、风控）中，XGBoost 和随机森林依然是最优解。不要因为深度学习火就跳过传统 ML。
