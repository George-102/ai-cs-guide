# CMU 17-313: Foundations of Software Engineering

## 课程简介

- 来源：Carnegie Mellon University
- 先修要求：编程基础
- 编程语言：Java
- 课程难度：🌟🌟🌟
- 预计学时：50 小时 +

CMU 17-313 是 CMU 的软件工程基础课程，覆盖软件工程的完整生命周期：需求分析、设计、实现、测试、部署和维护。

## 课程资源

- 课程网站：[cmu-313.github.io](https://cmu-313.github.io/)

## 课程特点

- **全面覆盖**：从需求分析到部署维护
- **项目驱动**：通过实际项目学习软件工程实践
- **团队协作**：强调团队合作和代码审查

## 自学建议

**做所有的项目。** 课程的核心是项目实践。

**学习代码审查。** 课程强调代码审查，这是软件工程师的重要技能。

## 备注

CMU 17-313 是软件工程入门的优秀课程。如果你想系统学习软件工程的完整流程，这门课是最佳选择。
