# CMU 17-313: Foundations of Software Engineering

## Descriptions

- Offered by: Carnegie Mellon University
- Prerequisites: Programming basics
- Programming Language: Java
- Difficulty: 🌟🌟🌟
- Class Hour: 50+

CMU 17-313 is CMU's foundational software engineering course, covering the complete software lifecycle: requirements, design, implementation, testing, deployment, and maintenance.

## Resources

- Website: [cmu-313.github.io](https://cmu-313.github.io/)

## Features

- Comprehensive coverage: from requirements to deployment
- Project-driven: learn SE practice through real projects
- Team collaboration: emphasizes teamwork and code review

## Tips

Do all projects. They are the core of the course.

CMU 17-313 is an excellent course for systematic study of the complete software engineering process.
