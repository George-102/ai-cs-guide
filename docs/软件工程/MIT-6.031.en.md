# MIT 6.031: Software Construction

## Descriptions

- Offered by: MIT
- Prerequisites: Python programming basics
- Programming Language: Java
- Difficulty: 🌟🌟🌟
- Class Hour: 60+

MIT 6.031 is MIT's software engineering course, focused on **writing high-quality, maintainable code**. Covers static checking, testing, specifications, ADTs, concurrency, version control.

## Resources

- Website: [web.mit.edu/6.031](https://web.mit.edu/6.031/www/sp21/)
- Notes: 30 readings covering all core topics

## Features

- 30 readings covering static checking, testing, specs, ADTs, concurrency, version control
- Accompanying exercises for each reading
- Java-based but concepts apply to all languages

## Tips

Read all materials in order. Each builds on the previous.

## Notes

MIT 6.031 is one of the best courses for learning to write "elegant" code.
