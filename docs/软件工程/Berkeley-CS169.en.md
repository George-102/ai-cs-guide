# Berkeley CS169: Software Engineering

## Descriptions

- Offered by: UC Berkeley
- Prerequisites: CS61B (Data Structures) or equivalent
- Programming Language: Ruby on Rails
- Difficulty: 🌟🌟🌟
- Class Hour: 60+

Berkeley CS169 is Berkeley's software engineering course, focused on **agile development and SaaS architecture**. The course project requires building a complete SaaS application with Ruby on Rails.

## Resources

- Website: [cs169.org](https://cs169.org/)

## Features

- Agile development: Scrum, TDD, continuous integration
- SaaS architecture: REST API, microservices, deployment
- Large project: Build a complete SaaS application

## Tips

Learn Ruby on Rails basics first.

## Notes

CS169 is the best course for learning agile development and SaaS architecture.
