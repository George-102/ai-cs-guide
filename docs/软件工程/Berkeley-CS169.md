# Berkeley CS169: Software Engineering

## 课程简介

- 来源：UC Berkeley
- 先修要求：CS61B（数据结构）同等水平
- 编程语言：Ruby on Rails
- 课程难度：🌟🌟🌟
- 预计学时：60 小时 +

Berkeley CS169 是 Berkeley 的软件工程课程，侧重于**敏捷开发和 SaaS 架构**。课程项目要求你用 Ruby on Rails 构建一个完整的 SaaS 应用。

## 课程资源

- 课程网站：[cs169.org](https://cs169.org/)

## 课程特点

- **敏捷开发**：Scrum、TDD、持续集成
- **SaaS 架构**：REST API、微服务、部署
- **大项目**：学期项目要求构建一个完整的 SaaS 应用

## 自学建议

**先学 Ruby on Rails 基础。** 课程用 Ruby on Rails 实现项目。

**做学期项目。** 课程的核心是学期项目，要求你构建一个完整的 SaaS 应用。

## 备注

CS169 是学习敏捷开发和 SaaS 架构的最佳课程。如果你想学习如何构建和部署 Web 应用，这门课是首选。
