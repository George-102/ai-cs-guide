# Foundation Models Overview

This module covers three foundational areas that CS students need to master in the AI era:

- **Introduction to Large Language Models**: Transformer architecture, pre-training, fine-tuning, RLHF — understand why LLMs "can talk"
- **Open-Source Model Ecosystem**: HuggingFace Hub, model selection, local deployment — from "can use APIs" to "can ship systems"
- **Multimodal Foundations**: Vision-language models, image generation, video understanding — AI is not just text

Learning path: Start with Introduction to LLMs to build a holistic understanding, then go deep on one area based on your interests. If you want to build RAG and Agents, LLM fundamentals are essential. If you want to work on game AI or visual applications, multimodal foundations are more valuable.
