# 操作系统

## 什么是操作系统

操作系统（Operating System，OS）是管理计算机硬件和软件资源的系统软件。它是应用程序和硬件之间的中间层，负责让多个程序共享 CPU、内存、磁盘和网络等资源，而不需要每个程序自己去管理这些底层细节。

你每天都在和操作系统打交道 — Windows、Linux、macOS 都是操作系统。你点击一个程序图标，操作系统负责加载它到内存、分配 CPU 时间、处理文件读写。你打开浏览器上网，操作系统负责管理网络连接。这些看似简单的事情背后，是操作系统在协调几十个甚至上百个并发任务。

## 为什么要学操作系统

**理解你写的程序到底在干什么。** 你调用一个 `printf`，背后涉及系统调用、用户态和内核态的切换、缓冲区管理、设备驱动。不理解操作系统，你写的程序就是个黑盒子 — 能跑，但不知道为什么能跑，也不知道出了问题怎么排查。

**写出更高效的程序。** 理解进程调度、内存分页、缓存机制，你就能写出对系统更友好的代码。比如，知道磁盘 I/O 是瓶颈，你就会减少不必要的文件操作；知道内存页面大小是 4KB，你就能更好地组织数据结构。

**面试中的硬通货。** 操作系统是计算机考研和大厂面试的核心科目。进程与线程的区别、死锁的条件、虚拟内存的原理 — 这些问题出现的频率极高。

**为后续学习打基础。** 计算机网络、数据库、分布式系统，这些高阶课程都建立在操作系统概念之上。不学好 OS，后面会处处碰壁。

## 核心知识模块

### 进程与线程

- **进程**是程序的一次执行实例，拥有独立的地址空间
- **线程**是进程内的执行单元，同一进程的线程共享地址空间
- 进程间通信（IPC）：管道、消息队列、共享内存、信号量
- 线程同步：互斥锁、条件变量、读写锁

### 内存管理

- **虚拟内存**：每个进程有自己的虚拟地址空间，操作系统负责映射到物理内存
- **分页与分段**：内存管理的两种基本方式
- **页面置换算法**：LRU、FIFO、Clock 等，决定内存不够时淘汰哪些页面
- **内存分配**：堆和栈的区别，malloc 的底层实现

### 文件系统

- 文件的组织方式：目录树结构
- 磁盘调度算法：SCAN、C-SCAN、SSTF
- 文件系统的实现：inode、日志文件系统

### I/O 与设备管理

- I/O 模型：阻塞、非阻塞、多路复用（select/poll/epoll）
- 设备驱动的角色
- 中断处理机制

### 并发与同步

- 临界区问题
- 经典同步问题：生产者-消费者、读者-写者、哲学家就餐
- 死锁的四个必要条件和预防策略

## 怎么学

**先建立整体框架。** 不要一上来就啃细节。先理解操作系统的核心职责（管理 CPU、内存、磁盘、设备），然后逐个击破。

**边学边做实验。** 操作系统是实践性很强的课程。光看书不写代码，概念永远停留在纸面上。推荐做 xv6 实验 — MIT 的教学操作系统，代码量小，结构清晰，非常适合入门。

**画图辅助理解。** 进程状态转换、内存地址映射、磁盘调度 — 这些概念用文字解释很抽象，画出来就清楚了。

**不要跳过 Linux 命令。** 学操作系统不是纯理论。`ps`、`top`、`strace`、`lsof` 这些工具能让你看到操作系统的实际行为。

## 推荐资源

**教材**

- [Operating Systems: Three Easy Pieces](https://pages.cs.wisc.edu/~remzi/OSTEP/) — 免费在线教材，讲解清晰，适合入门
- 《操作系统概念》（Operating System Concepts，恐龙书）— 经典教材，内容全面
- 《现代操作系统》（Andrew Tanenbaum）— 另一本经典，偏重设计原理

**课程**

- [MIT 6.S081](https://pdos.csail.mit.edu/6.828/2023/) — MIT 操作系统课程，基于 xv6，有完整实验
- [清华操作系统](https://www.xuetangx.com/course/THU08091000267/5883104) — 中文课程，讲解系统
- [UC Berkeley CS162](https://cs162.org/) — Berkeley 的操作系统课，有视频和作业

**实践**

- [xv6](https://pdos.csail.mit.edu/6.828/2023/xv6.html) — MIT 教学操作系统，代码量约 1 万行
- [Write your own OS](https://github.com/cfenollosa/os-tutorial) — 从零写操作系统的教程
