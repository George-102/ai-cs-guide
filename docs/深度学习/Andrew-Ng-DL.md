# 吴恩达 Deep Learning Specialization

## 课程简介

- 来源：Stanford / Coursera（Andrew Ng）
- 先修要求：机器学习基础、Python、线性代数
- 编程语言：Python（TensorFlow）
- 课程难度：🌟🌟🌟🌟
- 预计学时：50 小时 +

吴恩达的 Deep Learning Specialization 是全球最受欢迎的深度学习入门课程。5 门课从神经网络基础到 CNN/RNN，系统覆盖深度学习核心。

## 课程资源

- 课程网站：[Coursera](https://www.coursera.org/specializations/deep-learning)
- 课程笔记：可在课程网站下载

## 课程内容

- 第 1 课：神经网络基础
- 第 2 课：超参数调优、正则化、优化
- 第 3 课：结构化机器学习项目
- 第 4 课：卷积神经网络（CNN）
- 第 5 课：序列模型（RNN、LSTM）

## 自学建议

**做所有的编程作业。** 课程用 Python/TensorFlow 实现，不调高级 API。

**不要跳过数学推导。** Andrew Ng 的数学推导非常清晰。

**配合 Kaggle 练习。** 学完一个算法，去 Kaggle 上找对应的数据集练手。

## 备注

这门课是 DL 入门的最佳选择。如果你只想学一门 DL 课程，选这门就够了。
