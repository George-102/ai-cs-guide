# 深度学习（花书）

## 教材简介

- 来源：Ian Goodfellow、Yoshua Bengio、Aaron Courville
- 先修要求：机器学习基础、线性代数、概率论、微积分
- 课程难度：🌟🌟🌟🌟
- 预计学时：80 小时 +

《深度学习》（花书）是深度学习领域的"圣经"，由三位深度学习领域的顶级研究者编写。书名中的"花"来自封面上的花朵图案。

这本书覆盖了深度学习的完整知识体系，从数学基础到前沿研究。

## 教材资源

- 官方网站：[deeplearningbook.org](https://www.deeplearningbook.org/)
- 免费在线阅读：完全免费
- 中文翻译：《深度学习》（人民邮电出版社）

## 内容结构

- 第一部分：数学基础（线性代数、概率论、数值计算）
- 第二部分：深度学习基础（MLP、CNN、RNN、正则化）
- 第三部分：前沿研究（生成模型、表示学习）

## 自学建议

**先学吴恩达课程再读花书。** 花书偏理论，建议先建立直觉再深入理论。

**重点关注前 5 章。** 前 5 章覆盖了 DL 的核心概念。

**做书后习题。** 每章都有习题，能帮你检验理解程度。

## 备注

花书是 DL 理论进阶的最佳教材之一。如果你想深入理解 DL 的数学基础，这本书是必读的。

但花书不适合零基础入门。建议先学吴恩达课程，再读花书。
