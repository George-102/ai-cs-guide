# 动手学深度学习（d2l.ai）

## 教程简介

- 来源：李沐（D2L 团队）
- 先修要求：Python 基础、线性代数、概率论
- 编程语言：Python（PyTorch）
- 课程难度：🌟🌟🌟
- 预计学时：60 小时 +

动手学深度学习（d2l.ai）是李沐编写的免费在线教程，用 PyTorch 实现深度学习的每个算法。这本书的最大特点是**代码即文档**——每个概念都有完整的 PyTorch 实现。

## 教程资源

- 官方网站：[d2l.ai](https://d2l.ai/)
- 在线阅读：完全免费
- GitHub：[d2l-ai/d2l-zh](https://github.com/d2l-ai/d2l-zh)（中文版）

## 内容结构

- 第一部分：深度学习基础（MLP、CNN、RNN）
- 第二部分：现代深度学习（注意力机制、Transformer）
- 第三部分：应用（NLP、CV、推荐系统）

## 自学建议

**跟着教程写代码。** 每章都有完整的 Jupyter Notebook，可以直接运行。

**不要只看不做。** 动手学深度学习的核心是"动手"。

**配合吴恩达课程使用。** 吴恩达讲理论，d2l 做实践。

## 备注

d2l.ai 是深度学习入门的最佳实践教程。如果你想用 PyTorch 实现每个 DL 算法，选这本就够了。
