# Stanford CS231n: 计算机视觉与深度学习

## 课程简介

- 来源：Stanford University（Fei-Fei Li、Andrej Karpathy 等）
- 先修要求：机器学习基础、Python、线性代数、微积分
- 编程语言：Python（NumPy、PyTorch）
- 课程难度：🌟🌟🌟🌟
- 预计学时：60 小时 +

CS231n 是 Stanford 的计算机视觉课程，也是全球最好的深度学习课程之一。课程覆盖 CNN、RNN、GAN、Transformer、目标检测、图像分割等。

## 课程资源

- 课程网站：[cs231n.stanford.edu](http://cs231n.stanford.edu/)
- 课程视频：[YouTube](https://www.youtube.com/playlist?list=PL3FW7Lu3i5JvHM8ljYj-zLfQRF3EO8sYv)
- 实验代码：可在课程网站下载

## 课程特点

- **实验驱动**：从零实现 CNN、RNN、GAN
- **内容前沿**：涵盖 Transformer、Vision Transformer
- **作业设计**：每个作业都有详细的指导和自动评分

## 自学建议

**做所有的作业。** CS231n 的作业是课程的核心，从零实现每个模型。

**先掌握 NumPy。** 课程的前几个作业用 NumPy 实现，不调深度学习框架。

**阅读课程笔记。** CS231n 的课程笔记写得非常好，是理解概念的关键。

## 备注

CS231n 是深度学习入门的最佳课程之一。如果你想深入理解 CNN 和计算机视觉，这门课是首选。
