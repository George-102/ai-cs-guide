# 深度学习

## 简介

- 来源：吴恩达《Deep Learning Specialization》（Coursera）、李宏毅《深度学习》、CS231n（Stanford）
- 先修要求：机器学习基础、Python、线性代数、微积分、概率论
- 编程语言：Python（PyTorch 为主，TensorFlow 为辅）
- 课程难度：🌟🌟🌟🌟
- 预计学时：50 小时 +

深度学习（Deep Learning）是当前 AI 革命的引擎。从 GPT 到 Stable Diffusion，从 AlphaFold 到自动驾驶——这些突破都建立在深度学习之上。深度学习本质上是机器学习的一个分支，用多层神经网络来学习数据的层次化表示。

如果说机器学习是 AI 的基础，那么深度学习就是 AI 的**核心武器**。理解深度学习，你才能真正理解大模型、多模态 AI、生成式 AI 的底层原理。

## 核心知识模块

### 神经网络基础

- **感知机与多层感知机（MLP）**
- **反向传播算法**：链式法则、计算图
- **激活函数**：ReLU、Sigmoid、Tanh、GELU、SwiGLU
- **损失函数**：交叉熵、MSE、Hinge Loss
- **优化算法**：SGD、Momentum、Adam、AdamW
- **正则化**：Dropout、Batch Normalization、Weight Decay
- **权重初始化**：Xavier、He 初始化

### 卷积神经网络（CNN）

- **卷积操作**：卷积核、步长、填充
- **池化**：Max Pooling、Average Pooling
- **经典架构**：LeNet、AlexNet、VGG、GoogLeNet、ResNet
- **现代架构**：EfficientNet、ConvNeXt
- **应用场景**：图像分类、目标检测、图像分割

### 循环神经网络（RNN）

- **RNN 基础**：时间展开、梯度消失/爆炸
- **LSTM**：门控机制、遗忘门/输入门/输出门
- **GRU**：LSTM 的简化版
- **应用场景**：序列建模、时间序列预测

### Transformer 架构

- **自注意力机制（Self-Attention）**：Query、Key、Value
- **多头注意力（Multi-Head Attention）**
- **位置编码**：正弦位置编码、RoPE
- **Transformer 编码器-解码器架构**
- **BERT**：预训练 + 微调范式
- **GPT 系列**：自回归语言模型
- **ViT**：Vision Transformer，将 Transformer 用于视觉

### 生成模型

- **自编码器（AE）和变分自编码器（VAE）**
- **生成对抗网络（GAN）**：生成器、判别器、训练技巧
- **扩散模型（Diffusion Models）**：DDPM、Stable Diffusion
- **Flow-based 模型**

### 训练技巧

- **学习率调度**：Warm-up、Cosine Decay、ReduceLROnPlateau
- **数据增强**：图像翻转、裁剪、Mixup、CutMix
- **混合精度训练**：FP16/BF16
- **分布式训练**：数据并行、模型并行、流水线并行
- **梯度累积**：模拟更大 batch size

## 怎么学

**先学吴恩达，再读 d2l。** 吴恩达的课程帮你建立整体框架，d2l.ai 帮你用 PyTorch 实现每个算法。两者结合，理论和实践都不落下。

**用 PyTorch 从零实现每个模型。** 不要只调高级 API。从零写一次 MLP、CNN、RNN、Transformer，你会对这些架构有完全不同的理解。

**做 CS231n 实验。** 这门课的实验要求你用 NumPy 从零实现 CNN，是深度学习入门的最佳实践。

**复现经典论文。** 选一篇经典论文（如 ResNet、Transformer），尝试复现其核心实验。这是从"会用"到"精通"的关键一步。

## 备注

深度学习是理解大模型、多模态 AI、生成式 AI 的基础。即使你现在主要用现成的模型和 API，理解底层原理也能让你在遇到问题时做出更好的判断。

PyTorch 是当前学术界和工业界的主流框架，建议优先学习。TensorFlow 在企业级部署中仍有大量使用，可以作为补充。
