# Andrew Ng Deep Learning Specialization

## Descriptions

- Source: Stanford / Coursera (Andrew Ng)
- Prerequisites: ML basics, Python, linear algebra
- Programming Language: Python (TensorFlow)
- Difficulty: 🌟🌟🌟🌟
- Class Hour: 50+

Andrew Ng's Deep Learning Specialization is the most popular DL course. 5 courses from neural network basics to CNNs/RNNs.

## Resources

- Website: [Coursera](https://www.coursera.org/specializations/deep-learning)

## Content

- Course 1: Neural Network Basics
- Course 2: Hyperparameter Tuning, Regularization, Optimization
- Course 3: Structured ML Projects
- Course 4: CNNs
- Course 5: Sequence Models (RNN, LSTM)

## Tips

Do all programming assignments. The course uses Python/TensorFlow without high-level APIs.

## Notes

This is the best choice for DL beginners. If you only take one DL course, take this one.
