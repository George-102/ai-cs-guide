# Dive into Deep Learning (d2l.ai)

## Descriptions

- Source: Mu Li (D2L Team)
- Prerequisites: Python basics, linear algebra, probability
- Programming Language: Python (PyTorch)
- Difficulty: 🌟🌟🌟
- Class Hour: 60+

Dive into Deep Learning (d2l.ai) is a free online tutorial by Mu Li, implementing every DL algorithm in PyTorch. Its hallmark: code as documentation.

## Resources

- Website: [d2l.ai](https://d2l.ai/)
- GitHub: [d2l-ai/d2l-zh](https://github.com/d2l-ai/d2l-zh)

## Content

- Part 1: DL Foundations (MLP, CNN, RNN)
- Part 2: Modern DL (Attention, Transformer)
- Part 3: Applications (NLP, CV, Recommendations)

## Tips

Write code along with the tutorial. Each chapter has complete Jupyter Notebooks.

## Notes

d2l.ai is the best hands-on tutorial for DL beginners. If you want to implement every DL algorithm in PyTorch, this is the one.
