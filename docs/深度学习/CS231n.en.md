# Stanford CS231n: Computer Vision and Deep Learning

## Descriptions

- Source: Stanford University (Fei-Fei Li, Andrej Karpathy, et al.)
- Prerequisites: ML basics, Python, linear algebra, calculus
- Programming Language: Python (NumPy, PyTorch)
- Difficulty: 🌟🌟🌟🌟
- Class Hour: 60+

CS231n is Stanford's computer vision course and one of the best DL courses. Covers CNN, RNN, GAN, Transformer, object detection, segmentation.

## Resources

- Website: [cs231n.stanford.edu](http://cs231n.stanford.edu/)
- Lectures: [YouTube](https://www.youtube.com/playlist?list=PL3FW7Lu3i5JvHM8ljYj-zLfQRF3EO8sYv)

## Features

- Lab-driven: Implement CNN, RNN, GAN from scratch
- Cutting-edge: Covers Transformer, Vision Transformer
- Well-designed assignments with auto-grading

## Tips

Do all assignments. They are the core of the course.

## Notes

CS231n is one of the best courses for DL beginners. Essential for understanding CNNs and computer vision.
