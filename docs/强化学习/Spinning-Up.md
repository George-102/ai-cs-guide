# OpenAI Spinning Up in Deep RL

## 教程简介

- 来源：OpenAI
- 先修要求：深度学习基础、Python
- 编程语言：Python（PyTorch / TensorFlow）
- 课程难度：🌟🌟🌟
- 预计学时：30 小时 +

Spinning Up 是 OpenAI 推出的深度强化学习入门指南，包含教程、代码实现和关键论文推荐。

## 教程资源

- 官方网站：[spinningup.openai.com](https://spinningup.openai.com/)
- GitHub：[openai/spinningup](https://github.com/openai/spinningup)

## 内容结构

- RL 基础（MDP、策略、价值函数）
- 核心算法实现（VPG、PPO、DDPG、TD3、SAC）
- 关键论文推荐

## 自学建议

**跟着教程跑通每个算法的代码。** Spinning Up 的代码实现简洁明了，适合学习。

**先学 David Silver 课程。** Spinning Up 假设你已经了解 RL 基础。

**配合 Gymnasium 使用。** Spinning Up 的代码可以在 Gymnasium 环境中运行。

## 备注

Spinning Up 是深度 RL 入门的最佳实践指南。如果你想快速上手深度 RL 算法，选这个就够了。
