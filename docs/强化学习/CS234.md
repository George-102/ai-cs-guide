# Stanford CS234: Reinforcement Learning

## 课程简介

- 来源：Stanford University（Emma Brunskill）
- 先修要求：机器学习基础、Python、概率论
- 编程语言：Python
- 课程难度：🌟🌟🌟🌟
- 预计学时：50 小时 +

Stanford CS234 是 Stanford 的强化学习课程，由 Emma Brunskill 教授主讲。课程覆盖 RL 的核心算法和理论。

## 课程资源

- 课程网站：[web.stanford.edu/class/cs234](https://web.stanford.edu/class/cs234/)
- 课程视频：[YouTube](https://www.youtube.com/playlist?list=PLoROMvodv4rOSOPzutgyCTapiGlY2Nd8u)

## 课程内容

- MDP 和规划
- 值函数方法（DQN）
- 策略梯度方法（REINFORCE、PPO）
- 模型基方法
- 探索与利用
- 逆 RL 和 RLHF

## 自学建议

**做所有的编程作业。** 课程用 Python 实现 RL 算法。

**先学 David Silver 课程。** CS234 假设你已经了解 RL 基础。

**阅读课程笔记。** Emma Brunskill 的课程笔记写得非常好。

## 备注

CS234 是 RL 进阶的优秀课程。如果你想深入学习 RL 理论和算法，这门课是最佳选择。
