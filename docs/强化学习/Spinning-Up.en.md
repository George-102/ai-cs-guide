# OpenAI Spinning Up in Deep RL

## Descriptions

- Source: OpenAI
- Prerequisites: Deep learning basics, Python
- Programming Language: Python (PyTorch / TensorFlow)
- Difficulty: 🌟🌟🌟
- Class Hour: 30+

Spinning Up is OpenAI's deep RL getting-started guide, with tutorials, code implementations, and key paper recommendations.

## Resources

- Website: [spinningup.openai.com](https://spinningup.openai.com/)
- GitHub: [openai/spinningup](https://github.com/openai/spinningup)

## Content

- RL Foundations (MDP, Policy, Value Functions)
- Core Algorithm Implementations (VPG, PPO, DDPG, TD3, SAC)
- Key Paper Recommendations

## Tips

Run through each algorithm's code. Spinning Up's implementations are clean and great for learning.
