# Stanford CS234: Reinforcement Learning

## Descriptions

- Source: Stanford University (Emma Brunskill)
- Prerequisites: ML basics, Python, probability
- Programming Language: Python
- Difficulty: 🌟🌟🌟🌟
- Class Hour: 50+

Stanford CS234 is Stanford's RL course, taught by Professor Emma Brunskill. Covers core RL algorithms and theory.

## Resources

- Website: [web.stanford.edu/class/cs234](https://web.stanford.edu/class/cs234/)
- Lectures: [YouTube](https://www.youtube.com/playlist?list=PLoROMvodv4rOSOPzutgyCTapiGlY2Nd8u)

## Content

- MDPs and Planning
- Value Function Methods (DQN)
- Policy Gradient Methods (REINFORCE, PPO)
- Model-Based Methods
- Exploration and Exploitation
- Inverse RL and RLHF

## Tips

Do all programming assignments.

## Notes

CS234 is an excellent advanced RL course. Best choice for deep study of RL theory and algorithms.
