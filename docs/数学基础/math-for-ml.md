# Mathematics for Machine Learning

## 教材简介

- 来源：Marc Peter Deisenroth、A. Aldo Faisal、Cheng Soon Ong（Imperial College London）
- 先修要求：高中数学基础
- 课程难度：🌟🌟🌟
- 预计学时：80 小时 +

《Mathematics for Machine Learning》是一本专门为机器学习编写的数学教材。它覆盖了机器学习所需的全部数学基础：线性代数、微积分、概率论、优化理论。

这本书的最大特点是：**所有数学内容都围绕机器学习应用展开**。你不会学到"无用"的数学，每个概念都能在 ML 中找到对应的应用。

## 内容结构

### 第一部分：数学基础
- 线性代数（向量、矩阵、线性变换、特征值）
- 解析几何（距离、投影、超平面）
- 矩阵分解（SVD、特征分解、Cholesky 分解）

### 第二部分：机器学习中的数学
- 向量微积分（梯度、方向导数、Hessian 矩阵）
- 概率论与分布（贝叶斯推断、最大似然估计）
- 优化理论（梯度下降、凸优化、约束优化）

### 第三部分：应用
- 主成分分析（PCA）
- 高斯混合模型
- 线性回归
- 支持向量机（SVM）

## 自学建议

**先学线性代数和微积分基础，再读这本书。** 这本书假设你已经了解基本的线性代数和微积分概念。

**重点关注第一部分和第二部分。** 这些内容是机器学习的核心数学基础。

**做书后习题。** 每章都有习题，能帮你检验理解程度。

**配合其他课程使用。** 这本书适合作为 MIT 18.06（线性代数）和 MIT 18.01（微积分）的补充读物。
